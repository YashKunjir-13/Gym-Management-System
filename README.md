# Gym Management System

Technology –
•	Frontend : HTML , CSS, JavaScript, Bootstrap
•	Backend : python 
•	Database : SQLite 
•	Framework : Django

Introduction
The Gym Management System is designed to automate and simplify the daily operations of a fitness center. The project aims to provide a centralized digital solution for managing gym memberships, trainers, payments, attendance, and scheduling.

Through this system, the admin can monitor member registrations, track fee payments, assign trainers, and generate reports. Trainers can manage workout schedules and member progress, while members can view their plans, attendance, and renewal reminders.

This system helps reduce manual record-keeping, minimizes errors, and enhances communication between gym staff and members.
